Points on a sphere
------------------

You're given an array of [x,y,z] coordinates. Find out minimal
number of spheres needed to place each point onto a sphere
which has a center at [0,0,0].

Example 1
```
Input: [[1,1,1], [1,1,1]]
Ouput: 1
```
Explanation: Both points lay on a sphere with radius 1.
